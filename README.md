# ML Classification Project - SMS Spam Detection

## About
SMS Spam Detection using Machine Learning - InternSpark AI Internship Task 1

## Environment Setup
- Platform: Google Colab
- Language: Python 3

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## Dataset
- SMS Spam Detection Dataset from Kaggle
- 5572 messages with ham/spam labels

## How to Run
1. Open Copy_of_Welcome_to_Colab.ipynb in Google Colab
2. Upload spam_sms.csv dataset to Colab files
3. Run all cells in order

## Models Compared
1. Logistic Regression
2. Random Forest Classifier

## Results
| Model | Accuracy | Precision | F1 Score |
|---|---|---|---|
| Logistic Regression | 96.41% | 99.11% | 84.73% |
| Random Forest | 97.76% | 100% | 90.91% |

## Conclusion
Random Forest is the best model with 97.76% accuracy!
